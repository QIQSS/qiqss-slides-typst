# QIQSS Presentation Slides Template

This typst template is made for the QIQSS research group in Universit√© de Sherbrooke.
It can serve as example for anyone not a member of the group.

This template was made using [touying](https://github.com/touying-typ/touying).
Refer to the example in the `template/` folder for the usage of the template.

## Local installation

To use locally you can simply download the assets and the `lib.typ` file and place them
in your project. You would then import the template as
```typst
#import "lib.typ": *
```

You can also use [this small CLI program](https://github.com/yalap13/typi) to install the
template in the typst local data directory. You can then import the template as
```typst
#import "@local/qiqss-slides-typst:0.2.0": *
```
